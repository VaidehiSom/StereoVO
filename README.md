# Stereo Visual Odometry using Kitty Dataset

## Running the code
### Build the file using
```
cd ~/ws
mkdir build
cd build
cmake ..
make
```
### Then to run on BAL the dataset given in problem-16-22106-pre.txt

```
cd ~/ws
bin/run_kitti_stereo
```
